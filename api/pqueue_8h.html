typedef int(* pqueue_walk_callback_t) (pqueue_t *pq, uint64_t prio, void *value, void *priv)

Callback called for each node when walking the priority queue.

Parameters
pqA valid pointer to an initialized pqueue_t structure
prioThe priority of the current node
valueThe value stored in the current node
privThe private pointer passed to pqueue_walk()
Returns
1 If the walker can go ahead visiting the next node, 0 if the walker should stop and return -1 if the current node should be removed and the walker can go ahead -2 if the current node should be removed and the walker should stop

Create a new priority queue.

Parameters
modeThe operational mode of the priority queue
  • PQUEUE_MODE_HIGHEST : highest key have highest priority
  • PQUEUE_MODE_LOWEST : lowest key have highest priority
sizeThe maximum size of the queue (maximum number of elements that can be stored in the queue)
free_value_cbthe callback to use when an value has been removed from the queue and the value can be released
Returns
a newly allocated and initialized priority queue
