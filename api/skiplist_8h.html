Remove an item from the skip list.

Parameters
sklA valid pointer to an initialized skiplist_t structure
keyThe key of the item in the list where to store the new value
klenThe size of the key
valueIf not NULL the address of the value hold by the removed item will be stored at the memory pointed by the 'value' argument. If NULL and a free_value_callback is set, the value hold by the removed node will be released using the free_value_callback
Returns
0 if the item has been found and correctly removed; -1 otherwise
