typedef int(* avlt_walk_callback_t) (avlt_t *tree, void *key, size_t klen, void *value, void *priv)

Callback called for each node when walking the tree.

Parameters
treeA valid pointer to an initialized avlt_t structure
kThe key of the node where to store the new value
ksizeThe size of the key
valueThe new value to store
privThe private pointer passed to either avlt_walk() or avlt_walk_sorted()
Returns
1 If the walker can go ahead visiting the next node, 0 if the walker should stop and return -1 if the current node should be removed and the walker can go ahead -2 if the current node should be removed and the walker should stop
