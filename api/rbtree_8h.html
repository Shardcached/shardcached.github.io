typedef rbt_walk_return_code_t(* rbt_walk_callback) (rbt_t *rbt, void *key, size_t klen, void *value, void *priv)

Callback called for each node when walking the tree.

Parameters
rbtA valid pointer to an initialized rbt_t structure
keyThe key of the node where to store the new value
klenThe size of the key
valueThe new value to store
privThe private pointer passed to either rbt_walk() or rbt_walk_sorted()
Returns
RBT_WALK_CONTINUE If the walker can go ahead visiting the next node,
RBT_WALK_STOP if the walker should stop and return
RBT_WALK_DELETE_AND_CONTINUE if the current node should be removed and the walker can go ahead
RBT_WALK_DELETE_AND_STOP if the current node should be removed and the walker should stop
